# dtelecom-Airdrop-
Guide
1. What dTelecom is
dTelecom is a decentralized telecom and AI‑communication layer where users earn $DTEL tokens by using its apps (dMeet, livestreaming, AI Voice Agent, Spatial) and completing tasks. 
A significant portion of the token supply is allocated to early users via a points‑based airdrop / rewards program, with points later converting to $DTEL at TGE. 
2. Main airdrop platforms
You’ll farm on multiple platforms; the same real‑usage activity is usually counted across all. 
Zealy – main campaign hub (social + onboarding tasks). 
Galxe – NFT‑based quests and deeper engagement. 
TaskOn / QuestN – task‑specific quests and identity‑based activities. 
Native apps – dMeet (video calls), Livestreaming, AI Voice Agent, Spatial rooms. 
Referral system – invite users via your ref link to earn extra points.
